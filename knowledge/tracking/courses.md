# Learning Resources & Courses

> Last updated: 2026-03-06

## Structured Learning Path

### Phase 1: Foundations
1. **Andrej Karpathy — Neural Networks: Zero to Hero**
   - From backprop basics to GPT, code from scratch
   - [karpathy.ai/zero-to-hero.html](https://karpathy.ai/zero-to-hero.html)

2. **Andrej Karpathy — Deep Dive into LLMs** (YouTube)
   - Bottom-up principles + practical usage guide

3. **Sebastian Raschka — LLMs from Scratch**
   - Build ChatGPT-level LLM step by step in PyTorch, 17hr video
   - [GitHub](https://github.com/rasbt/LLMs-from-scratch)

### Phase 2: Comprehensive LLM Engineering
4. **Maxime Labonne — LLM Course** (70K+ stars)
   - Three parts: Fundamentals / The LLM Scientist / The LLM Engineer
   - Covers pretraining, SFT, alignment, fine-tuning (Llama 3.1 + Unsloth)
   - [GitHub](https://github.com/mlabonne/llm-course) | [HuggingFace](https://huggingface.co/blog/mlabonne/llm-course)

5. **Karpathy — LLM101n**
   - Build Storyteller AI in Python/C/CUDA from scratch
   - [GitHub](https://github.com/karpathy/LLM101n)

### Phase 3: Advanced Topics
6. **fast.ai — Practical Deep Learning Part 2** (Jeremy Howard)
   - 30+ hours, includes Stable Diffusion from scratch
   - [course.fast.ai](https://course.fast.ai/)

7. **Stanford CS324: Large Language Models**
   - Modeling, theory, ethics, systems

8. **HuggingFace Ultrascale Playbook**
   - Companion to Nanotron, large-scale training guide

### Phase 4: Staying Current
9. **Sebastian Raschka — Ahead of AI** (Substack, 150K+ readers)
   - [magazine.sebastianraschka.com](https://magazine.sebastianraschka.com/)

10. **Lilian Weng — Lil'Log** (OpenAI)
    - Deep analysis posts ("like mini research papers")
    - [lilianweng.github.io](https://lilianweng.github.io/)
