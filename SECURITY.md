# Security Policy

## Scope

This repository contains knowledge content (LaTeX books, markdown articles) and automation scripts (GitHub Actions, Python). There is no deployed web service or API.

## Reporting a Vulnerability

If you discover a security issue in our automation scripts or GitHub Actions workflows, please report it responsibly:

1. **Do NOT open a public issue**
2. Email the maintainers or use [GitHub Security Advisories](https://github.com/ZenAlexa/llm-observatory/security/advisories/new)
3. Include a description of the vulnerability and steps to reproduce

We will acknowledge your report within 48 hours and provide a timeline for a fix.

## Supported Versions

| Component | Supported |
|-----------|-----------|
| GitHub Actions workflows | ✅ Latest on `main` |
| Python scripts | ✅ Latest on `main` |
| Book content (LaTeX) | N/A (not a security surface) |
